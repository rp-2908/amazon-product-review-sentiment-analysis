# Amazon Product Review Sentiment Analysis

End-to-end NLP project performing sentiment analysis on real Amazon customer reviews.

## 📋 Project Overview
- **Dataset**: 1,597 Amazon product reviews
- **Goal**: Classify reviews as Positive or Negative
- **Techniques**: Text preprocessing, TF-IDF Vectorization, Classification models

## 📥 Dataset
- **Source**: [Amazon Product Reviews Dataset](https://www.kaggle.com/datasets/yasserh/amazon-product-reviews-dataset) by Yasser H.
- **Size**: ~1,597 reviews
- **How to use**: Download the dataset from Kaggle and place `7817_1.csv` in the project folder.

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- NLTK (text preprocessing)
- Scikit-Learn (TF-IDF, Logistic Regression, Random Forest, LinearSVC)
- Matplotlib & Seaborn (visualization)

## 📊 Results
- **Best Model**: Logistic Regression (with class balancing)
- **Accuracy**: **73.8%**
- **F1-Score**: 0.74 (Weighted)
