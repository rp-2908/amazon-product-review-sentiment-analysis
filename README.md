# Amazon Product Review Sentiment Analysis

**End-to-end NLP project** performing sentiment analysis on real **Amazon customer reviews** using TF-IDF and Scikit-Learn.

### 📋 Project Overview
- **Dataset**: 1,597 Amazon product reviews
- **Goal**: Classify reviews as Positive or Negative
- **Techniques**: Text Preprocessing, TF-IDF Vectorization, Classification

### 🛠️ Technologies Used
- **Python**, Pandas, NumPy
- **NLTK** (Text Preprocessing)
- **Scikit-Learn** (TF-IDF, Logistic Regression, Random Forest, LinearSVC)
- Matplotlib & Seaborn (Visualization)

### 📊 Results
- **Best Model**: Logistic Regression (with class balancing)
- **Accuracy**: **73.8%**
- **F1-Score**: 0.74 (Weighted)

### 📥 Dataset
Download from: [Kaggle - Amazon Product Reviews](https://www.kaggle.com/datasets/yasserh/amazon-product-reviews-dataset)

### 📁 Key Learnings
- End-to-end ML pipeline for text data
- Feature extraction using TF-IDF
- Handling imbalanced datasets
- Model training, evaluation & comparison
